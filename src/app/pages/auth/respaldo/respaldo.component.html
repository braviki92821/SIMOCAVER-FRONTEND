Crea o inserte los datos de respaldo del sistema

Instrucciones

1.-Para los datos respaldados se descargara un archivo .json que contiene las fechas de pronostico registradas

2.-Asegurese de realizar una copia de manera semanal o mensual para garantizar la actualidad de los datos

3.-Estos datos solo son sacados de la base de datos por lo cual las imaganes deberan descargarse de manera individual

4.-Las imagenes tambien deberan subirse de manera individual

5.-Esta archivo.json debera cargarse a travez de el administrador de la base de datos(MongoDb)
